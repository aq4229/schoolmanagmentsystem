INSTRUCTIONS:

The pre-requisite to fill information in Academic Settings (Session, Branch, Class & Section) and Fee’s Setting (Fee items & Fee structure) in your SaaS account in order to generate the “Excel File” that include ID’s against respective fields.

  • Student type id: (the corresponding ID for ‘Admission Type name’ in the ‘Admission Type Data form’ needs to be entered here)
  • Fee Structure** id: (the corresponding ID for ‘Fee Structure name’ in the ‘Fee Structure Data form’ needs to be entered here)
  • Session id: (the corresponding ID for ‘Session Name’ from the ‘Session Data form’ needs to be entered here).
  • Branch** id: (the corresponding ID for ‘Branch Name’ from the ‘Branch Data form’ needs to be entered here).
  • Class** id: (the corresponding ID for ‘Class name’ from the ‘Class Data form’ needs to be entered here).
  • Section id: (the corresponding ID for ‘Section name’ from the ‘Section Data form’ needs to be entered here).
  • While entering data into "Gender" and "Primary Contact" columns, make sure you capitalize (upper case 'A') the first alphabet (e.g "Father", "Mother", "Guardian" and "Male/Female") .

IMPORTANT:

  • Fields with ** (Asterick) are ‘mandatory fields’, hence, required for Software to enter student data.
  • If you will not enter the Roll no. and Registration no. then the system will generate Roll no. and Registration no. by itself.
  • Log in as an administrator if your SaaS account only allows people with administrator privileges to upload data.
  • Use the following browsers; Windows Explorer, Google Chrome or Mozilla/Netscape's Download Manager, navigate to the “MW SMS MS Excel file” you downloaded. Double-click to open the file.

Bulk Upload

After Entering Student Data in the Excel sheet, save file as ‘CSV File’ , as system only accept ‘CSV file’ format.
